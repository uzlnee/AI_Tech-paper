# 7. Contribution 하는 방법

## 7.1 Fork를 한다.
- 원본 저장소를 내 계정으로 복사(Fork)

## 7.2 PR를 보낸다.
- Fork한 저장소에서 변경 내용을 작업한 뒤 Pull Request(Pull Request)를 생성

## 7.3 Approve를 받으면 Merge!
- 리뷰 후 승인을 받으면 원본 저장소에 변경사항이 병합(Merge)

>이 문서는 계속 업데이트 예정입니다.
궁금한 사항이나 제안이 있다면 Issue 혹은 Pull Request로 알려주세요!
